Should be running in http://localhost:61452

## Building the code
* Make sure you have latest preview of [.NET Core SDK 5](https://www.microsoft.com/net/download/core) installed
* Clone
* `cd src\JustSending`
* `dotnet run` or `dotnet watch run`
* Should be running in http://localhost:61452
